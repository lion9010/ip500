Módulo de radio inalámbrico IP500® 868 MHz

El CNX100 fue diseñado para cumplir con los requisitos de una solución estándar inalámbrica abierta y segura para la automatización de edificios, casas inteligente y la seguridad inalámbrica. Junto con el microcontrolador integrado y una amplia gama de periféricos, el CNX100 usa la mejor tecnología RF de sub-GHz de su clase para cumplir con los requisitos de alcance de larga distancia, el alto rendimiento de la tasa de datos y las expectativas de seguridad. El CNX100 cumple con la última IEE802.15.4g-2011 y ETSI EN300 220-1 y FCC47 CFR Sección 15.247. Una batería con protocolos IP500 y la interfaz de la aplicación BACnet permiten una fácil integración en los sistemas de los clientes mediante el uso de una interfaz API estándar.

Características Claves

  • Estructura de datos Nativa al protocolo IP500® protocol
  • Frecuencia de operación de sub GHz
  • Soporta IPv6
  • IEEE 802.15.4
  • 6LoWPAN
  • Connecciones: UART, Antena
  • Soporte de Aplicación BACnet
  • Dimensiones compactas: 15 mm x 34 mm
  • Red en malla
  • Certificado CE y ETSI

Adecuado para las siguientes aplicaciones

  • Edificios y casas comerciales inteligentes.
  • automatización inalámbrica
  • medición inteligente
  • redes de sensores de baja potencia
  • gestión inteligente de energía ecológica
  • Seguridad inalámbrica
