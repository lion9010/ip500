CoreNetix – IP500® Kit de Desarrollo R3

El kit de desarrollo CNX - IP500® R3 ofrece todo lo necesario para que los desarrolladores de sistemas integren sus productos de Edificios Inteligentes en una red IP500®. Aprenda lo fácil que es controlar y comunicarse con sensores y actores a través de una interfaz web gráfica. En menos de 10 minutos, puede tener una red de demostración IP500® con todas las funciones configuradas en su entorno. Simplemente encienda el Netbook, coloque los tableros DK y enciéndalos, y en unos momentos puede probar las funciones en el tablero. El módulo CNX100 montado en las placas de desarrollo con todas las funciones proporciona la base para el despliegue de una red IP500®. Se proporciona una demostración de desarrollo como marco para crear sus propias aplicaciones para sus propios dispositivos compatibles con IP500®

El KIT incluye

  • Tres tableros CNX DK
  • Tablero para Puerto R2
  • Netbook
  • Demo de Inicio Rápido
  • Demo de Desarrollo
  • CAble USB
  • Documentación
