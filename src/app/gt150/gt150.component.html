Administrador de puerto inalambrico IP500®

GT150 proporciona la puerta de enlace con todas las funciones entre la red de sensores y otras redes o dispositivos informáticos. Cumple con las últimas IEE802.15.4g-2012 y ETSI TS 102 887-1. Junto con el enrutamiento del software IP500, permite la recopilación de datos desde la red de sensores y el envío de comandos de configuración a nodos individuales o grupos más grandes. El CNX GT150 puede manejar hasta 5000 sensores dentro de una red. Sin embargo, para construir redes más grandes, el GT150 se puede agrupar.

Key Features

  • Estructura de datos de protocolo Nativo IP500®
  • Frecuencia de operación Sub GHz
  • Gestión de dispositivos y redes inalambricas
  • IEEE 802.15.4
  • 6LoWPAN
  • Actaulización remota del firmware
  • Integración rapida y fácil de usar de APIs
  • Gestión de claves para la comunicación inalámbrica.
  • Integración a los sitemas Host via TCP/IP
