Puerto Web- MICROSENS IP500®

El MICROSENS IP500®-Ethernet Gateway está diseñado para interconectar redes de sensores inalámbricos basadas en IP500® con las soluciones del sistema de construcción inteligente de MICROSENS. El módulo del puerto de enlace compacta se puede conectar directamente a un puerto de enlace descendente MICROSENS Micro Switch. Para simplificar los procesos de administración de servicios de TI, el puerto de enlace de internet-IP500® consta de LED separados para monitorear la disponibilidad de voltaje de CC en la placa y el latido del corazón del sistema interno. Durante el registro y el análisis de la causa raíz de los incidentes, el estado de estos LED permite que un sistema de escritorio analice y clasifique facilmente los problemas clasifique fácilmente los problemas que pueden estar relacionados con el suministro de energía o el estado del sistema.

Características Claves

  • Verdadera cobertura de banda dual para sub GHz y bandas de 2,4 GHz ISM
  • Antenas internas, conectores de antena externos opcionales
  • Suministro a través del puerto de enlace ascendente de Ethernet con PoE (PD), IEEE 802.3af
  • LED de control independiente para indicar el ritmo del sistema y el voltaje de CC a bordo
  • Configuración y gestión a través de Micro Switch MICROSENS adjunto.
  • Soporte de procesos de gestión de servicios.
  • Fácil de instalar en cualquier sistema de instalación eléctrica estándar de 45 mm
  • El factor de forma y de perfil son idénticos a los Micro Switches de MICROSENS
