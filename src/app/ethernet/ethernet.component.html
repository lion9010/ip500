MICROSENS IP500®-Ethernet Gateway

The MICROSENS IP500®-Ethernet Gateway is designed to interface IP500®-based wireless sensor networks to MICROSENS Smart Building System solutions. The compact gateway module can be directly attached to a MICROSENS Micro Switch downlink port. In order to simplify IT Service Management processes the IP500®-Ethernet Gateway consists of separate LEDs for monitoring the availability of DC on-board voltage and the internal system heart-beat. During recording and root cause analysis of incidents the status of these LEDs enables a Service Desk Analyst to easily classify issues which may be related to the provisioning of power or the system status.

Key Features

  • True Dual Band coverage for Sub GHz and 2.4 Ghz ISM Bands
  • Internal antennas, optional external antenna connectors
  • Supply through Ethernet uplink port with PoE (PD), IEEE 802.3af
  • Independent Control LED′s to indicate system heartbeat and DC on-board voltage
  • configuration and management through attached MICROSENS Micro Switch
  • Support of Service Management processes
  • Easy to install into any standard 45 mm electrical installation system
  • Form factor and shape are identical with MICROSENS Micro Switches

Suitable for the following applications

  • smart commercial buildings and homes
  • wireless automation
  • smart metering
  • low power sensor networks
  • intelligent eco energy management
  • wireless security
